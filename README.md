# MONTO FOOD

Site vitrine responsive pour le restaurant Monto Food, réalisé avec Bootstrap 5.

## Fonctionnalités
- Page d'accueil avec hero, présentation, spécialités, contact et footer
- Design inspiré d'une maquette Figma
- Responsive (mobile, tablette, desktop)

## Installation
1. Cloner le dépôt
2. Installer les dépendances :
   ```bash
   npm install
   ```
3. Ouvrir `home.html` dans votre navigateur

## Structure du projet
- `home.html` : page principale
- `src/` : images et ressources
- `node_modules/` : dépendances Bootstrap

## Dépendances
- [Bootstrap 5](https://getbootstrap.com/)
